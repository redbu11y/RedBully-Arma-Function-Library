# Readme

These scripts are made for the Arma group from The Few Good Men forums.
CBA_A3 and ACE are requirements for many Functions.

## Cargo Loading and Wreck Loading

    These two functions are made to allow for loading objects and wrecks into vehicles, primarily trucks. Cargo Loading uses the vanilla arma ViV system to load vehicles and objects into the ViV cargo. Wreck Loading allows for wrecks to be put on trucks, this system is hardcoded and ViV does not allow for wrecks to be loaded.
